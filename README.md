TVWorks Kodi Repository
Repositório de add-ons para Kodi.
Como Instalar
Passo 1: Adicionar Fonte

Abre o Kodi
Vai a Definições > Gestor de ficheiros
Clica em Adicionar fonte
Onde diz "Nenhum", introduz o seguinte URL:

   https://tinyurl.com/tvworks

Dá um nome à fonte (por exemplo: "TVWorks")
Clica em OK

Passo 2: Instalar Repositório

Volta ao ecrã inicial do Kodi
Vai a Add-ons
Clica no ícone da caixa aberta (canto superior esquerdo)
Seleciona Instalar de um ficheiro zip
Escolhe a fonte que criaste ("TVWorks")
Seleciona o ficheiro .zip do repositório
Aguarda a notificação de instalação

Passo 3: Instalar Add-ons

Clica novamente no ícone da caixa aberta
Seleciona Instalar do repositório
Escolhe o repositório TVWorks
Navega pelas categorias e instala os add-ons que quiseres


Nota: Certifica-te que tens a opção "Fontes desconhecidas" ativada nas definições do Kodi antes de instalar.
